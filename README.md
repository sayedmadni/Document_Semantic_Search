# Document Semantic Search 

A simple application meant to assist college students in getting guidance for their college admissions. 

We have a mid size corpus of about 1000 documents related to admissions. We use a few key components for this project. 

1. Dockling - https://github.com/docling-project/docling
2. Chonkie - https://github.com/chonkie-inc/chonkie?tab=readme-ov-file
3. Qdrant - https://qdrant.tech/documentation/quickstart/
4. GitHub - https://github.com/sayedmadni/Document_Semantic_Search
5. Streamlit - https://streamlit.io/playground
6. LLM - https://ollama.com/library/llama3.1
